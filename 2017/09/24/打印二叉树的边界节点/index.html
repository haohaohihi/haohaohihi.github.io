打印二叉树的边界节点

发表于 2017-09-24 | 分类于 数据结构与算法 |

【题目】：

　　给定一颗二叉树的头节点 head，按照如下两种标准分别实现二叉树边界节点的逆时针打印。

　　标准一：

　　1、头节点为边界节点；

　　2、叶节点为边界节点；

　　3、如果节点在其所在的层中是最左或者最右的，那么也是边界节点。

# 二叉树节点的定义
class Node(object):
    def __init__(self, value=-1, left=None, right=None):
        self.value = value
        self.left = left
        self.right = right

# 递归法获得二叉树的深度
def getHeight(head, l):
    # head类型: Node
    # l类型： int
    if head == None:
        return l
    return (Max(getHeight(head.left, l+1), getHeight(head.right, l+1)))

# 获得二叉树每一层最左和最右的节点
# 思路：先将每次的节点分别赋给左右边，先进行左递归，故左边界已经满足条件，
#       然后进行右递归，左边界不变，赋值后右边界满足条件
def setEdgeMap(head, l, edgeMap):
    if head == None:
        return None
    if edgeMap[l][0] == None:
        edgeMap[l][0] = head
    else:
        edgeMap[l][0] = edgeMap[l][0]
    edgeMap[1][1] = head
    setEdgeMap(head.left, l+1, edgeMap)
    setEdgeMap(head.right, l+1, edgeMap)

# 打印不在最左最右边的边界节点
def printLeafNotInMap(head,l, edgeMap):
    if head == None:
        return None
    if (head.left == None and head.right == None and head != edgeMap[l][0] and head != edgeMap[l][1]):
        print(head.value)
    printLeafNotInMap(head.left, l+1, edgeMap)
    printLeafNotInMap(head.right, l+1, edgeMap)

##### 打印所有边界节点（包括头节点、叶节点、每层的最左最右节点）
def printEdge(head):
    if head == None:
        return None
    height = getHeight(head, 0)
    # 用一个二维列表存储每一层最左、最右节点，注意此处初始化列表的方法
    edgeMap = [[None for i in range(2)] for i in range(height)]
    # 打印左边界节点
    for i in range(height):
        print(edgeMap[i][0].value)

    # 打印不在最左最右边的边界节点
    printLeafNotInMap(head, 0, edgeMap)

    # 打印右边界，但不是左边界的节点
    for i in range(height-1, -1, -1):
        if edgeMap[i][1] != edgeMap[i][0]:
            print(edgeMap[i][1])
