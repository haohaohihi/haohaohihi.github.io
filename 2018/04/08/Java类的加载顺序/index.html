Java类的加载顺序

发表于 2018-04-08 | 分类于 Java |

Java类的加载顺序为：

  1. 父类的静态代码块（包括静态初始化块、静态变量，但不包括静态方法）
  2. 子类的静态代码块（包括静态初始化块、静态变量，但不包括静态方法）
  3. 父类的非静态代码块（包括非静态初始化块、非静态属性）
  4. 父类构造函数
  5. 子类的非静态代码块（包括非静态初始化块、非静态属性）
  6. 子类构造函数

其中，(1)(2)在实例化对象之前就会加载，并且是按照代码顺序加载。
